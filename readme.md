# 🔐 Password Security Checker

A beginner-friendly Password Security Checker built using Python and Streamlit.

## Features

- ✅ Check password strength
- ✅ Detect weak password patterns
- ✅ Generate secure passwords
- ✅ User-friendly web interface
- ✅ Beautiful blue-themed UI

## Technologies Used

- Python
- Streamlit
- Regex (re)
- Random
- String

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Password-Security-Checker.git
```

Go to the project folder:

```bash
cd Password-Security-Checker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
streamlit run app.py
```
